# 🔐 Password Guessing Simulator (Brute Force Demo)

## 📌 Overview

This project is a simple Python-based simulator that demonstrates how a **brute-force password guessing attack** works.

The program repeatedly generates random guesses until it matches the user-provided password.

> ⚠️ This project is for **educational purposes only**.

---

## 🚀 Features

* Simulates brute-force password guessing
* Random password generation using digits
* Demonstrates inefficiency of weak passwords
* Beginner-friendly Python implementation

---

## 🧠 Concepts Used

* Loops (`while`, `for`)
* Random number generation
* String manipulation
* Basic system commands

---

## ▶️ How to Run

1. Make sure Python is installed
2. Clone the repository or download the file
3. Run the script:

```bash
python crakingpassward.py
```

4. Enter a numeric password when prompted
